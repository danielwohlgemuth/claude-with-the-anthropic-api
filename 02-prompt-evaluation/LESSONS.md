# Prompt Evaluation Lessons
- The evaluation process has three steps: dataset generation, model execution, and results analysis
- The dataset can also include the expected output and evaluation criteria
- Grading the output can be done by a model (non-deterministic) or programmatically (deterministic) to get a final score
- When grading by a model, including a reasoning field can act like a "thinking" mode to produce more grounded results
- It's best to start with a simple prompt and adjust based on the score
