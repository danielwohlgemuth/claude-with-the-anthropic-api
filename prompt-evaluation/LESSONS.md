# Prompt Evaluation Lessons
- The evaluation process has three steps: dataset generation, model run, and results analysis
- When generating the dataset, it can also include the expected output and evaluation criteria
- When grading the output, some of it can be done by a model (non-deterministic) and some of it programmatically (deterministic) to get a final score
- When grading by a model, including a reasoning field can act like a "thinking" mode to produce more grounded results
- It's best to start with a simple prompt and adjust based on the score
